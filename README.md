# perntest1



## Description

One command to create a full stack PERN application.  

### Dependencies
* [Node.js](https://nodejs.org/en/) 14 or higher 
* [PostgreSQL](https://www.postgresql.org/download/windows/) 13 or higher pre-installed 




### Executing program
```
npx perntest1

cd server

nodemon index

cd client 

npm run start

npx perntest1
```
