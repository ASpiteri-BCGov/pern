# perntest1



## Description

One command to create a full stack PERN application.  

### Dependencies
* Node 14 or higher
* PostgreSQL pre-installed




### Executing program
```
npx perntest1

cd server

nodemon index

cd client 

npm run start

npx perntest1
```
